# tennis-motion
iOS app to collect high-frequency tennis motion data from its Apple Watch counterpart, written in Swift using new CoreMotion capaibilities (mainly CMBatchedSensorManager). 

The development process behind this app is described here: https://hackernoon.com/getting-hiqh-frequency-tennis-motion-data-from-apple-watch

The early data anaysis can be found here: https://hackernoon.com/getting-hiqh-frequency-tennis-motion-data-from-apple-watch-part-2
